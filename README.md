# Yu-Gi-Oh! Twitter Bot üÉè

A Python bot that tweets a random Yu-Gi-Oh! card each time it runs. It fetches card data from the [YGOProDeck API](https://db.ygoprodeck.com/api-guide/) and posts the card name, type, description, and image to Twitter.

---

## Prerequisites

- Python 3.8 or higher  
- A Twitter Developer Account with **Read and Write** permissions
